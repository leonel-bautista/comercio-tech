Backend - Rest API endpoints

    GET Obtener productos (sin autenticación).

  • GET /api/products

    Devuelve la lista completa de productos.

  • GET /api/products/:id

    Devuelve un producto específico.

    POST Crear productos (requiere autenticación).

  • POST /api/products/create

    Crea y guarda un nuevo producto en la base de datos.

    Headers requeridos: Authorization Bearer <token> Body: 
    {
  "nombre": "Laptop Gaming",
  "descripcion": "Laptop de alta gama",
  "precio": 1500000,
  "stock": 10,
  "imagen": "<url>",
  "categorias": ["electrónica", "computadoras"]
}

    DELETE Eliminar productos (requiere autenticación).

  • DELETE /api/products/:id

    Elimina un producto específico de la base de datos.

    Headers requeridos: Authorization Bearer <token>

Autenticación

Para crear o eliminar productos, necesitas un token JWT válido.

Realiza un request POST a /auth/login con tus credenciales para obtener un token. Luego, incluye ese token en el header Authorization: Bearer <token> en las peticiones protegidas.

  • Body: 
    {
  "email": "<email>",
  "password": "<password>"
}
